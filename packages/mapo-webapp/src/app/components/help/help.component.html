Add a node:

  • Press 't' to select the 'text' tool, then left-click on the canvas
  • Type in node content
  • Press 'Escape' or left-click on the canvas to finish
Tip: You can also double-click on the canvas to add a node without pressing 't'

Edit a node

  • Double-click on a node to begin editing its content
  • Press 'Escape' or left-click on the canvas to finish

Draw an edge:

  • Press 'e' to select the 'edge' tool
  • Left-click on the start node
  • Left-click on the end node

Label an edge:

  • Double-click on any existing edge
  • Enter text
  • Press 'Escape' or left-click on the canvas to finish

Delete objects:

  • Press 'Escape' to select the pointer tool
  • Left-click the node or edge you want to delete
  • Press "Backspace", "Delete", or "d"

Move the canvas:

  • Right-click and drag on the canvas

Zoom the canvas:

  • Scroll up to zoom in
  • Scroll down to zoom out
